## 🚀 로컬 개발

이 블로그를 로컬에서 실행하려면:

```bash
# Jekyll 설치 (Ruby 필요)
gem install jekyll bundler

# 의존성 설치
bundle install

# 로컬 서버 실행
bundle exec jekyll serve
```
